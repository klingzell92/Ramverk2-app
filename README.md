Ramverk2 app
=================================

Installera
--------------
Börja med att clona repot härifrån Github. Och kör efter det:

```shell
npm install
```
För att köra alla tester så kör man:

```shell
npm test
```

För att köra enhetstesterna i docker så använder man:

```shell
npm run test1 test2 test3
```
